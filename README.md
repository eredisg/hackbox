# Hackbox

This Vagrant script will spin up the latest version of Kali with my included dotfiles and software for CTFs.

## Setup
Download Vagrant from [here](https://www.vagrantup.com/)

```
vagrant up
```

## Software used/downloaded
* Alacritty Terminal
* Starship.rs
* Tmux
* Burp Suite Pro
* Jetbrains Toolbox
